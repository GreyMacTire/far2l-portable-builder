# Сборка портативной версии far2l

Инструкция по созданию автономного (standalone) дистрибутива файлового менеджера far2l.

## Конфигурация

**Режим: GUI|WX / TTY|X / TTY**

**Плагины:**
   ```bash
   -DUSEWX=yes -DARCLITE=no -DNETROCKS=no
   ```

## Шаги сборки

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/elfmz/far2l.git
   ```

2. **Перейдите в каталог проекта:**
   ```bash
   cd far2l
   ```

3. **Переключитесь на нужный тег (например, v_2.8.0):**
   ```bash
   git checkout v_2.8.0
   ```

4. **Запустите скрипт сборки:**
   ```bash
   ./build_standalone.sh
   ```

## Результат
После успешного завершения процесса готовый портативный дистрибутив будет находиться в директории:
`far2l/portable`
